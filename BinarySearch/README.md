# 🔍 Binary Search Problems

This folder contains beginner to intermediate level Binary Search problems solved in C++.

Binary Search is an efficient searching algorithm with a time complexity of:

O(log n)

---

## 📚 Problems Solved

| Problem | Difficulty | Concepts Used |
|----------|------------|----------------|
| Search Insert Position | Easy | Classic Binary Search |
| First Bad Version | Easy | Binary Search on Answer |
| Sqrt(x) | Easy | Binary Search on Numbers |
| Guess Number Higher or Lower | Easy | Interactive Binary Search |
